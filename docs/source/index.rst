Genestack Tasks API Reference
=============================

Welcome to the Genestack Tasks API reference!
The Tasks API is a Python module used by Genestack initialization scripts to interact with the
Genestack system, i.e. working with Genestack files and metainfo, and performing Genestack storage I/O operations.

This library is used inside platform to execute your scripts.
You should not install this library locally in order to execute your scripts inside the Platform.
But you may download sources and add connect them to your IDE
for access to sources, documentation and to have :ref:`AUTOCOMPLETE`.
